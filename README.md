# boiler-gains

## Objective
Web application that help Purdue students reach their nutrition goals with dining hall food.
- Show basic nutritional data for each food
- Fit dietary needs such as being vegetarian/gluten free
- Track how much much of each macro and micro you have consumed
- Compare it to goals
- Categorize food both by macro/micro in each dining hall
- Sort macro/micro by most to least

Basically Purdue Menus + Goal-Oriented help

## Basic Design
- frontend
    - React + Tailwind CSS
    - one main page
    - dining hall pages
    - meal logging page
    - about & nutritional goals page
    - account page
    - etc...
- backend
    - PostgreSQL
    - Python FastAPI
    - keep track of users, food items, dining halls, nutritional goal options, and meal logs
